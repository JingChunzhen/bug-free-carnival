# 位运算
## 1. N &= N-1 
形式等同于 N &= (N - 1) 将bit位中最后一个为1的位取反为0，leecode 461 

## 2. 子集运算
A & B -> AB 交集

A | B -> AB 并集

A ^ B -> AB 对称差 

^ 异或运算，1^0 == 1, 1^1 == 0, 0^0 == 0, leetcode 461, 136(Single Number系列)

可以使用位运算求全部的子集

set = {w, a, e, v, u}

|set| = 5 |subset| = 2^5 (0~31) 0表示空集 31(2^5-1)表示全集
```
for(i = 0 -> 31)
{
     for(j=0 -> 5) i & (1 << j)
}
```
